# Claude Code Software Teams

Claude Code icin hazir yazilim takimi agent tanimlari. Iki ayri takimdan olusur:

## Mobile App Team

Cross-platform mobil uygulama gelistirme takimi.

**Tech Stack:** Expo + React Native + NativeWind + TypeScript

| Agent | Rol | Dosya |
|-------|-----|-------|
| `mobile-lead` | Team Lead | `.claude/agents/mobile-lead.md` |
| `mobile-dev` | Mobile Developer | `.claude/agents/mobile-dev.md` |
| `mobile-backend` | Backend Developer | `.claude/agents/mobile-backend.md` |
| `mobile-designer` | UI/UX Designer | `.claude/agents/mobile-designer.md` |
| `mobile-qa` | QA Engineer | `.claude/agents/mobile-qa.md` |

## Landing Page Team

| Agent | Rol | Dosya |

## Kullanim

1. Bu repo'yu projene klonla veya `.claude/agents/` klasorunu kopyala
2. Claude Code'da agent'lari kullanmaya basla

Agent dosyalarini ihtiyacina gore ozellestir - her `.md` dosyasi ilgili agent'in rolunu, kurallarini ve tech stack'ini tanimlar.
